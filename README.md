### Installation
### Resquest
- Node.js > 13.x

#### Steps:
- clone the repository: ```git clone https://github.com/garciamendes/Telzin.git```
- In **Telzin** folder run:
```npm install```

#### Running your project
- In Telzin run: ```npm start```
